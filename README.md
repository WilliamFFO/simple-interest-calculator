# Simple Interest Calculator

## Description
This is a simple interest calculator that computes interest using the formula:

**Simple Interest = (Principal × Rate × Time) / 100**

## Formula
- **Principal (P)**: Initial amount of money
- **Rate (R)**: Annual interest rate (%)
- **Time (T)**: Time period in years

## Example
If Principal = $1000, Rate = 5%, Time = 2 years:
- Simple Interest = (1000 × 5 × 2) / 100 = $100
